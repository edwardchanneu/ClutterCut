# Project Overview — ClutterCut

**ClutterCut** is a cross-platform Electron desktop application that helps users automatically organize files in a selected directory using user-defined rules. The product's core value proposition is **transparency and reversibility** — users always preview changes before execution and can undo any run.

- **PRD:** `project_memory/clutter-cut-prd.md` (v1.0 MVP, authored by Heather Carminati & Edward Chan)
- **Scrum Issues:** `project_memory/clutter-cut-issues.md`
- **Current Phase:** MVP (Sprints 1–5)
- **Platforms:** macOS + Windows (no Linux for MVP)
